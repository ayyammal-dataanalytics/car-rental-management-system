# Car Rental Management System (Python OOP Project)

A Python console-based Car Rental Management System designed using Object-Oriented Programming (OOP) concepts.  
This project simulates a car rental service where users can view available cars, rent them, return them, and calculate rental price efficiently.

---

## 🚗 Project Overview

This system manages car rentals using Python classes. It allows users to:

- View available cars  
- Rent a car by entering its registration number  
- Return a rented car  
- Calculate total rental cost based on number of days  
- Maintain car availability status dynamically  

The project demonstrates the use of:

- Python Classes  
- Objects  
- Methods  
- Encapsulation  
- Conditional Logic  
- Basic OOP Principles  

---

## 🛠 Features

- Add and manage multiple cars  
- Rent a car and mark it as unavailable  
- Return a car and update availability  
- Display only available cars  
- Calculate rental price (Price per day × Number of rental days)  
- Handle invalid car numbers  
- Prevent renting already rented cars  

---

## 💰 Rental Price Calculation

The system calculates the rental bill using:

```
Total Cost = Price per Day × Number of Days
```

Example:

If car price per day = ₹1500  
Days rented = 3  

```
Total = ₹4500
```

This ensures accurate billing and real-world rental simulation.
